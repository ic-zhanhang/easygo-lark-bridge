# 集成指南：EasyGo + feishu-cursor-claw

全部路径在 **`workspace/easygo-lark-bridge/`** 内，无需 `~/tools`。

## 目录结构

```
easygo-lark-bridge/
├── claw/       feishu-cursor-claw
├── runtime/    Agent workspace + logs/inbox/sessions
├── config/
└── scripts/
```

## 安装

```bash
cd /Users/ic/workspace/easygo-lark-bridge
cp config/easygo.env.example config/easygo.env
# 填写 ALLOWED_OPERATOR_OPEN_ID、FEISHU_APP_SECRET、CURSOR_API_KEY

bash scripts/install.sh
```

若曾使用 `~/tools/feishu-cursor-claw` 或 `~/workspace/easygo-claw`，`install.sh` 会自动迁移。

## 运行

```bash
# 前台
cd claw && bun run server.ts

# 开机自启
bash scripts/claw-service.sh install
bash scripts/claw-service.sh logs
```
